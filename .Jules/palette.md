## 2024-05-22 - Backend UX is Developer Experience
**Learning:** When working on a backend-only repository (e.g., WCF Service), the "User Interface" is the API definition. Improving Swagger/OpenAPI documentation (titles, descriptions, examples) is the equivalent of adding tooltips or labels in a frontend.
**Action:** For backend services, look for `Swagger` or `OpenAPI` libraries and enhance the metadata attributes `[Description]`, `[SwaggerWcfPath]`.
